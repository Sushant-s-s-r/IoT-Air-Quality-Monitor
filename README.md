
> ## ⚡ Quick Start

1. Clone this repo
```bash
   git clone https://github.com/Anubhacharchi/IoT-Air-Quality-Monitor.git
```
2. Open `src/air_quality_monitor.ino` in Arduino IDE
3. Install all required libraries
4. Update WiFi credentials and Blynk token
5. Select board: `ESP32 Dev Module`
6. Upload and open Serial Monitor at `115200` baud
7. ## 🤝 Contributing
Pull requests are welcome! For major changes, please open an issue first.
